# Retail Hero Research

Research project on uplift modeling and causal inference for personalized marketing in the X5 Retail Hero competition.

The project investigates machine learning approaches for estimating heterogeneous treatment effects and optimizing customer targeting strategies in retail recommendation systems.

## Objectives
- Predict incremental customer response to marketing actions
- Compare classical uplift-modeling approaches with neural-network-based methods
- Evaluate treatment effect estimation quality using uplift metrics

## Implemented Methods

### Meta-Learners
- S-Learner
- T-Learner
- X-Learner

### Neural Approaches
- TARNet
- DragonNet

## Evaluation Metrics
- Qini AUC
- Uplift AUC
- uplift@K

## Features
- Exploratory data analysis (EDA)
- Customer behavior analysis
- Treatment/control group modeling
- Feature engineering for retail data
- Comparative benchmarking of uplift methods
- Statistical evaluation of uplift quality

## Tech Stack
- Python
- PyTorch
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Research Focus
The project explores practical applications of causal inference and uplift modeling in recommendation systems and targeted marketing optimization.
